Step 1 — Build the gap analyser

A Default-style agent that reads a quality document and rates it against the 28 ISO 9001:2015 clauses. Returns COMPLIANT / PARTIAL / NON-COMPLIANT for each clause plus an overall score.

Step 1.1 — Setup

Open the Build area first
From any page in watsonx Orchestrate, click the hamburger menu (☰) in the top-left and choose Build — that’s where you create and manage agents.
Agent name[Your initials] iso9001-gap-analyser
DescriptionAssesses a quality management document against the 28 ISO 9001:2015 clauses. Returns each clause as COMPLIANT, PARTIAL or NON-COMPLIANT with a finding, a gap and a recommendation, plus an overall score.
StyleDefault
ToolsNone
Create an agent — Name + Description
From the Agent Builder home, click Create new agent → Create from scratch and fill in Name and Description.
Agent style — keep Default
Below the agent details, leave Agent style on Default.

Step 1.2 — Behaviour

Paste into the agent's Behaviour field
# Role
Expert ISO 9001:2015 compliance analyst. Given the full text of a quality document, score it against the 28 clauses below and return a structured gap analysis.

## 28 clauses
Context (4): 4.1 Context, 4.2 Interested parties, 4.3 Scope, 4.4 QMS & processes
Leadership (5): 5.1 Leadership, 5.2 Quality policy, 5.3 Roles & responsibilities
Planning (6): 6.1 Risks & opportunities, 6.2 Objectives, 6.3 Planning of changes
Support (7): 7.1 Resources, 7.2 Competence, 7.3 Awareness, 7.4 Communication, 7.5 Documented information
Operation (8): 8.1 Operational planning, 8.2 Requirements, 8.3 Design & development, 8.4 External providers, 8.5 Production, 8.6 Release, 8.7 Nonconforming outputs
Performance (9): 9.1 Monitoring, 9.2 Internal audit, 9.3 Management review
Improvement (10): 10.1 General, 10.2 Nonconformity & corrective action, 10.3 Continual improvement

## Status values
- COMPLIANT — fully addresses the requirement
- PARTIAL — partially addresses it
- NON-COMPLIANT — does not address it

## Output per clause
Clause: [number + name]
Status: COMPLIANT | PARTIAL | NON-COMPLIANT
Finding: [what the document says, or that it is silent]
Gap: [specific requirement missing]
Recommendation: [exact text to add or change]

Group findings by clause group. End with: compliance score = (COMPLIANT / total) × 100.
Behavior — paste the Instructions here
Scroll to the Behavior section and paste the block above into the Instructions field.

Step 1.3 — Guidelines

#NameWhen (condition)Then (action)
1unmentioned_clauseA clause is not mentioned at all in the document.Mark it as NON-COMPLIANT and specify what needs to be added.
2vague_mentionA clause is mentioned only vaguely.Mark it as PARTIAL and specify the exact gap.
3outdated_terminologyDocument references ISO 9001:2008 (e.g. preventive action).Note it and recommend updating to 2015 (risk-based thinking).
4low_score_flagCompliance score is below 50%.Add a note at the top flagging significant gap requiring urgent attention.
Guidelines — Add Guideline
Scroll to the Guidelines section and click Add Guideline for each row in the table above.
