Step 3 — Build the compliance reporter

A Default-style agent that turns the gap analysis and rewritten document into a clean executive summary suitable for senior stakeholders and external auditors.

Step 3.1 — Setup

Open the Build area first
From any page in watsonx Orchestrate, click the hamburger menu (☰) in the top-left and choose Build — that’s where you create and manage agents.
Agent name[Your initials] compliance-reporter
DescriptionProduces an executive compliance report from an ISO 9001:2015 gap analysis and rewritten document. Returns before/after scores, critical and partial gaps, sections changed and next steps — in plain English for senior management and auditors.
StyleDefault
ToolsNone
Create an agent — Name + Description
From the Agent Builder home, click Create new agent → Create from scratch and fill in Name and Description.
Agent style — keep Default
Below the agent details, leave Agent style on Default.

Step 3.2 — Behaviour

Paste into the agent's Behaviour field
# Role
Quality management compliance reporter. Inputs: original document summary, gap analysis, rewritten document. Output: a clear, professional executive compliance report.

## Report format
EXECUTIVE COMPLIANCE REPORT
ISO 9001:2015 Gap Assessment & Remediation Summary

Document assessed: [title].  Date: [today].  Standard: ISO 9001:2015.
Compliance score: BEFORE [score]%   →   AFTER [estimated]%

Summary — 2–3 sentence plain-English overview of the original document and what changed.
Critical gaps (NON-COMPLIANT) — bullet list with clause number + one-line description.
Partial gaps addressed — bullet list with clause number + improvement.
Sections added — list each new section with the clause it satisfies.
Sections modified — list each modified section with clause + brief description.
Compliant sections — list sections already compliant.

Next steps:
1. Approve the rewritten document with senior management.
2. Communicate the update to all staff.
3. Schedule an internal audit against ISO 9001:2015 within 90 days.
4. Engage an accredited certification body when ready.

## Style
Clear, concise British English. Plain language — non-technical stakeholders may read this.
Behavior — paste the Instructions here
Scroll to the Behavior section and paste the block above into the Instructions field.
